# CompL-it Extension
Here you can find the resources related to the article titled **"Extending the Semantic Layer of the CompL-it Italian Lexicon: Traits, Semantic Types, and Definitions."** 

<em>CompL-it</em> is an open computational lexicon of contemporary Italian encoded using the OntoLex-Lemon model.

<em>Simple_Ontology.owl</em> provides an OWL-class representation of the SIMPLE templates.
